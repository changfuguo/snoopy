# transform.js

## Install

```bash
npm install --save transform.js
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'transform.js'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
